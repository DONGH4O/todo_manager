## 6. 验证限制

- 当前环境中的内置浏览器阻止直接访问本地 `file://` 页面。
- 使用临时 localhost 服务访问原型时，浏览器也返回 `net::ERR_BLOCKED_BY_CLIENT`。
- 因此本轮未产出自动截图；视觉验收以用户多轮人工审查确认为准。

## 7. M5 接力事项

M5 实现时应优先处理：

- 将设计系统 v2 映射到 `gui/theme.py` 或新的集中主题模块。
- 保持四态任务枚举与 `engine.models.VALID_STATUSES` 一致。
- 搜索下拉层级必须高于月历和详情面板。
- 搜索下拉必须支持点击搜索区域外关闭。
- 主题模式入口采用 `自动 / 明色 / 暗色` 三态分段控件，默认自动跟随系统。
- 今日节奏、任务详情、子任务列表、搜索结果必须内部滚动，避免 body 级滚动。
- 删除后 toast 显示撤销；普通保存和刷新不显示撤销。
- 子任务使用紧凑状态选择控件，保存时按所选四态写入状态。
- 所有按钮保留 tooltip。
- GUI 通过 engine API 改数据，不在 widget 中维护独立业务缓存。
